How to restart a recording with the ecube

Written 7/8/2020 - updated version of the ecube

SURE WATCHTOWER IS OFF AND NO LONGER SAVING THE CAMERA STREAMS

see the watchtower tab for how to do that

Is there a recording currently running?

  1. Click the powershell with the current recording running. (Should be open or minimized, black terminal window with lots of text in it.
  2. Press Ctrl+C until it stops (usually takes 2-3 tries). You'll know its done cause the command line will be back and youll see the working directory, like you're inside a folder and about to ls or something, rather than blank and taking commands.
  3. Unplug the ecube (big black brick on the power strip next to the computer on the rack)
  4. Wait 10 seconds
  5. Plug it back in
  6. Wait another 10 seconds

Starting a new recording

  1. Open the UDP_console (black icon on the desktop - one on the right - that says UDP_console)
  2. Hit "Scan" until an I.P. address pops up that doesn't end in 1
  3. Hit "Start"
  4. Open a powershell window
  5. Open the text file called "start up"
  6. cd into the location in the start up file (the first line of the file)
  7. look at the second line of the start up file, find the date in that file and make sure it's today's date (this is the log for the ecube)
  8. type the updated command from the second line of the start up file into the command line
  9. the command line should now look like its running a program, you can still type commands though
  10. typing "list" will show you what ports are availible - digital and analog are usually always there, make sure the headstage you want to connect is also there
  11. type "add {port #} {# of channels}" so if I was trying to add a headstage on port 7 with 3 boards I would type "add 7 192"
  12. do this for all animals you're trying to connect
  13. type "start" when you're done
  14. open the open ephys folder on the desktop
  15. find the open ephys application in that folder (just called open_ephys) and open it
  16. hit run when the pop up comes up
  17. drag down 'Ecube' and 'lfp' from the right
  18. click 'ip' in the orange ecube box at the bottom and click to the correct ip address that you connected to
  19. click 'headstages' in the drop down below the 'ip' dropdown
  20. on the right side of the small orange box make sure the correct port is highlighted, click the port until it shows the right number of channels
  21. click "ch+"
  22. in the lfp box click the sideways hat looking thing to launch that viewer
  23. up in the top right of the screen click the 'play' button - look at your data - admire it - or hate it - doesn't matter
  24. pause the data visualization (hit the play button again)
  25. click the backwards play button in the top right (this is the dropdown button)
  26. see those 3 rectangles? Those are naming boxes. In the far left one put the animal name ("CAF00001" e.g.). the middle box auto fills with the date
  27. on the far left of the screen click the dropdown and change it to "ecube hub"
  28. hit the record button on the right (the circle)
  29. Exit open ephys
  30. repeat the open ephys steps for every animal you're recording from
